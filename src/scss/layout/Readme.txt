Папка layout/ содержит всё, что принимает участие в постройке раскладки сайта или приложения. 
Эта папка может содержать стили для основных частей сайта (шапка, подвал, навигация, боковая панель…), сетка или даже CSS-стили для всех форм.

|– layout/
|   |– _navigation.scss   # Навигация
|   |– _grid.scss         # Сетка
|   |– _header.scss       # Шапка
|   |– _footer.scss       # Футер
|   |– _sidebar.scss      # Сайдбар
|   |– _forms.scss        # Формы
|   …                     # и так далее
